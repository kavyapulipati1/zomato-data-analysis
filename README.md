 # Zomato Data Analysis

This project involves analyzing the Zomato restaurant dataset using tools like Pandas, Matplotlib, and Seaborn.

## Features
- Data cleaning
- Exploratory Data Analysis (EDA)
- Visualizations
- Insights from cuisine, location, and rating patterns

## How to run
1. Clone this repo
2. Open `zomato-analysis.ipynb`
3. Run cells in Jupyter
